Guess my number is a simple game. You have to guess a number between 1 and 20. After each guess you get a hint if the number is too high, too low or correct.

Your score starts at 20 and decreases each time you guess wrong. When your score is 0, the game is over.

This project was built by following the online course "The Complete JavaScript Course 2021: From Zero to Expert!" by Jonas Schmedtmann. I modified the design a little bit and added the JavaScript parts.
