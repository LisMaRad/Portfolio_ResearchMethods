Studentenfutter is a community web app for students at the University of Applied Sciences in Salzburg to connect with other people while sharing a meal. Be the chef or be a guest â€“ you decide.

The web application is structured as follows:

  1. Registration: Users start by registering with their university accounts. After registration, they complete their profile where they input personal details such as location, favorite cuisine, allergies and a profile picture.
  2. Feed: The feed displays events for the current week. Users can apply filters based on location and date. Events are presented with key information and users can send inquiries directly to the host.
  3. Creating an event: Users can decide to host a dinner or lunch and create a new event for it. They input details such as the dish, number of attendees, cost, date/time and location.
  4. Invitations: If a user is interested in an event, they send a request to the host. The host then confirms these requests.

Additional Features:

  1. Users can provide ratings for attended events.
  2. Event ratings are visible to all users and an overall rating is displayed on user profiles.
  3. Various notifications are sent to users, displayed in the "Notifications" section, and sent via emails (e.g., "Your dinner event is scheduled for today at 5:00 PM").

This project was created together with my colleagues Kerstin Reichinger, Lisa Rader and Alexandra Buchecker during our MultiMediaTechnology course (6th semester) at the University of Applied Sciences Salzburg.
